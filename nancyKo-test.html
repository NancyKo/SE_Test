  • Q: Describe the differences between the IE6, IE7, and W3C box models.

    A: Box model is referring to dimensions of the element calculated by the Margin, border and padding. There are differences in how these dimensions are calculated and therefore pages are displayed differently in different browsers. Major differences between IE and W3C is how width is calculated. W3C calculates width by the amount of space the contents of the element takes up, whereas in IE the width is calculated by the amount of space the border around the contents occupy. There are several methods to address this. We could create separate stylesheets for each browser (which is not ideal). We can also work around this by using CSS 'border-box' sizing to resolve some cross-browser compatibilty, but that would not affect IE6. A better way to make the page consistent across browsers is using CSS filters that specify browsers and either show or hide certain properties and values based on the browsers support for W3C standards.
